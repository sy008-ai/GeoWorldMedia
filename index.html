⚡ 3-Step Claude AI Setup

1
Get Your Anthropic API Key
Go to console.anthropic.com → API Keys → Create Key. Copy the key (starts with sk-ant-...). Store it safely — you only see it once.
2
Deploy Cloudflare Worker Proxy
Go to workers.cloudflare.com → Create Worker. Copy the Worker code from inside the <script> tag comments at the top of this HTML file. Set environment variables: ANTHROPIC_API_KEY and ALLOWED_ORIGIN. Deploy.
3
Set Your Worker URL
Open this HTML file. At the top of the <script> block, find CFG.PROXY_URL: '' and replace the empty string with your Cloudflare Worker URL. Push to GitHub. Vercel redeploys in 30 seconds. All AI features activate instantly.
✅ Total setup time: ~45 minutes. After that, GeoWorldMedia runs fully autonomously — no human needed. AI fetches news, generates analysis, handles all user interactions, and updates 24/7.
